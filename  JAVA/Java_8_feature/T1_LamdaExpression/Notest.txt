What is Lambda Expression?
Ans: The lambda expression is a ananomas function or method. It is used to concise the code or reduce the size of code make it more interactive.

Step to make any function Lambda expression ?
The Lambda Expression has no return type.
The lambda Expression has no specifier.
The lambda expression has no name.
It has only arrow ()->.

Normal function :

Public int add(int a , int b )
{
return a+b;
}

Lambda expression 

(int a,int b ) -> { return a+b;}
